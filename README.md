# DEPLOY WORDPRESS WEBSITE ON AWS USING ELASTIC BEANSTALK 
## ABOUT THE PROJECT:
This project shows how a WordPress website can be deployed on AWS
using Elastic Beanstalk and Amazon RDS.
## SERVICES USED IN PROJECT:
- AWS EC2
- AWS ELASTIC-BEANSTALK
- AMAZON RDS (MYSQL)
- IAM
- VPC (DEFAULT)
## DEPLOYMENT STEPS:
- create an amazon RDS database
- create an elastic beanstalk environment
- deploy the application
- download and configure wordpress
- connect wordpress with RDS
- verify the website deployment
## PROJECT OUTCOME:
successfully deployed the wordpress website on aws using elastic beanstalk and amazon RDS
